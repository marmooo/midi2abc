# midi2abc

Convert MIDI to ABC notation by using Tone.js note sequence generated by
Magenta.js.

## Demo

- [tone2abc](https://marmooo.github.io/tone2abc/)
- [MIDI Score](https://marmooo.github.io/midi-score/)

## Build

```
bash build.sh
```
